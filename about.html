About TinyPlanner

You have friends over for the evening. You know what you're cooking but you don't know when to start making it?

Create a plan, enter the time you need to be finished, or ready by. Chuck in some steps to get things done and we'll tell you when you need to start.

No connection required

TinyPlanner is free to use and works without an internet connection. Go on, try it now! You only need to visit tinyplanner.com once to be able to use anywhere, even in space.

We hope you like it, and if you do, tell us at @tinyplanner

We're new, so if you find a problem please let us know.
